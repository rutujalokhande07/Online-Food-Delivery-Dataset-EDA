# 🍔 Online Food Delivery Data Analysis

This repository contains an Exploratory Data Analysis (EDA) project on the **Online Food Delivery Dataset** using Python. The project focuses on understanding customer behavior, demographics, feedback, and ordering patterns through data analysis and visualization.

## 📌 Project Objective

The main objective of this project is to:
- Explore the online food delivery dataset.
- Clean and analyze the data.
- Visualize customer trends.
- Generate business insights that can help improve online food delivery services.

## 📂 Dataset

- **Dataset:** Online Food Delivery Dataset
- **Source:** Kaggle
- **Rows:** 388
- **Columns:** 12

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab / Jupyter Notebook

## 📊 Analysis Performed

- Displayed the first 10 records
- Checked dataset shape
- Identified missing values
- Analyzed gender distribution
- Explored monthly income categories
- Found the top educational qualifications
- Compared gender across occupations
- Analyzed marital status and feedback
- Compared family size by monthly income
- Studied the relationship between age and family size
- Created a correlation heatmap
- Analyzed occupation, monthly income, and feedback
- Compared average family size by educational qualification
- Visualized customer locations using latitude and longitude
- Identified pin codes with the highest number of customers
- Summarized business insights from the data

## 📈 Key Business Insights

- Most customers are young adults.
- Most customers gave positive feedback.
- Some pin codes have a higher number of customers.
- Customer locations are concentrated in specific areas.
- Occupation and monthly income influence ordering behavior.
- Family size can affect food ordering patterns.
